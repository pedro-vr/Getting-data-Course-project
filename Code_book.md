## Getting and Cleaning Data - Course Project

## Changes

The original raw data was modified by the following:

Merging the training and test sets to create one only data set.

Extracting only the measurements on the mean and standard deviation for each measurement.

Applying descriptive activity names to name of the activities in the merged data set

Labelin the data set with descriptive variable names.

Creating a second tidy data set with the average of each variable for each activity and each subject.

## Descriptions

Identififiers

The first two columns of the data set (Subject and Activity) are the identifiers.

Subject: the ID of the Subject
Activity: the Name of the Activity performed
